# Star Rating

별점 평가 컴포넌트입니다. 읽기 전용/인터랙티브 모드를 지원합니다.

## Import

```tsx
import { StarRating } from '@choblue/ui/star-rating'
```

## 읽기 전용

<StarRatingReadOnly />

```tsx
<StarRating value={3} readOnly size="sm" />
<StarRating value={4} readOnly size="md" />
<StarRating value={5} readOnly size="lg" />
```

## 인터랙티브

별을 클릭해보세요:

<StarRatingInteractive />

```tsx
const [rating, setRating] = useState(0)

<StarRating value={rating} onChange={setRating} size="lg" />
```

## Props

| Prop | 타입 | 기본값 | 설명 |
|------|------|--------|------|
| `value` | `number` | - | 현재 별점 (필수) |
| `max` | `number` | `5` | 최대 별 개수 |
| `readOnly` | `boolean` | `false` | 읽기 전용 모드 |
| `onChange` | `(value: number) => void` | - | 값 변경 핸들러 (`readOnly=false` 시 필수) |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | 별 크기 |
