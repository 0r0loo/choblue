# Textarea

여러 줄 텍스트 입력 컴포넌트입니다.

## Import

```tsx
import { Textarea } from '@choblue/ui/textarea'
```

## Sizes

<TextareaSizes />

```tsx
<Textarea size="sm" placeholder="작은 입력" />
<Textarea size="md" placeholder="보통 입력" />
<Textarea size="lg" placeholder="큰 입력" />
```

<TextareaStates />

## Props

| Prop | 타입 | 기본값 | 설명 |
|------|------|--------|------|
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | 텍스트 크기 |
| `rows` | `number` | - | 줄 수 |
| `disabled` | `boolean` | `false` | 비활성화 |
| `aria-invalid` | `boolean \| string` | - | 유효성 검사 실패 표시 |
